# 👗 StyleAI — AI Fashion Stylist

An AI-powered web application that analyzes user photos and suggests personalized outfit recommendations.

## 🚀 Features
- Image Upload & Preview
- AI-based Outfit Suggestions
- Virtual Try-On (basic)
- Clean UI (HTML, CSS, JS)
- Client-side application (no backend)

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- Unsplash API (for images)
- (Optional) Claude AI API

## 📸 How It Works
1. Upload your photo
2. Click "Find My Look"
3. Get outfit suggestions instantly

## 🖥️ Run Locally

```bash
python -m http.server 3000

```
